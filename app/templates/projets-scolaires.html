Projet avec l'entreprise Take a Waste : Application web pour l'optimisation de la gestion des déchets

Novembre 2020 - Avril 2021

Mon équipe a obtenu le prix du meilleur projet entreprise de l'ESIEE pour ce projet.

Durant six mois, j'ai travaillé avec l'entreprise Take a waste pour leur permettre d'automatiser deux tâches :
- récupérer et nettoyer des données sur internet sur les flux de déchets
- visualiser les données via un tableau de bord

Pour le premier objectif, un logiciel de scraping automatique des données a été réalisé.
Concernant le second, une application web a été créée pour permettre au client de l'entreprise de visualiser leurs flux de déchets.

Vous pouvez retrouver le pitch présenté lors du concours des meilleurs projet ESIEE et une vidéo résumée du projet. Pour plus d'informations, n'hésitez pas à me contacter.

Technologies utilisées : Python, Flask, PostgreSQL, HTLM, CSS, Heroku, Zoho Analytics

Projet scolaire : Concours Kaggle

Avril 2021 - Mai 2021

L’objectif du concours Kaggle est de prédire la note sur dossier, obtenue par un candidat pour l’entrée dans une école d’ingénieur, à l’aide des données déjà fournies (moyennes, appréciations, …). Un ensemble d'entraînement était à notre disposition, il contenait le score à prédire et les données d’entrée des candidats, son but est de permettre de prédire les notes d’un ensemble de test en fonction des données fournies. Le classement Kaggle est effectué sur le RSME entre les scores prédits de l’ensemble de test et les scores réels (inconnus pour nous).
